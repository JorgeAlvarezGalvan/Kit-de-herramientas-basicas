1. Información del sistema
🔹 Programa: CPU-Z

Descripción:
CPU-Z es una herramienta gratuita que proporciona información detallada sobre el hardware del equipo, incluyendo CPU, memoria RAM, placa base y tarjeta gráfica. Es muy útil para diagnóstico rápido del sistema.

Descarga:
https://www.cpuid.com/softwares/cpu-z.html

2. Testeo de disco
🔹 Programa: CrystalDiskInfo

Descripción:
Permite monitorizar el estado de los discos duros y SSD mediante tecnología S.M.A.R.T., detectando fallos antes de que ocurran.

Descarga:
https://crystalmark.info/en/software/crystaldiskinfo/

3. Recuperación de datos
🔹 Programa: Recuva

Descripción:
Software gratuito que permite recuperar archivos eliminados accidentalmente de discos duros, USB o tarjetas de memoria.

Descarga:
https://www.ccleaner.com/recuva

4. Recuperación de contraseñas

🔹 Programa: John the Ripper

Descripción:
Herramienta de auditoría de seguridad que permite descifrar contraseñas mediante ataques de diccionario y fuerza bruta.

Descarga:
https://www.openwall.com/john/

5. Herramienta libre (útil)
🔹 Programa: Hiren’s BootCD PE

Descripción:
Un entorno de rescate completo que incluye múltiples herramientas para diagnóstico, recuperación de datos, eliminación de malware y mantenimiento del sistema.

Descarga:
https://www.hirensbootcd.org/
